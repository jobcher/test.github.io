一家位于日本大阪心斋桥的高级珠宝店7日下午被抢，一名店员被刺身亡。警方当天在关西机场将中国籍嫌疑人后驭波（男27岁）逮捕。
警方称，当时珠宝店的一名女性店员在为伪装成顾客的嫌疑人展示手表时，嫌疑人突然掏刀。另一名男性店员弘中辰弥（30岁）在阻止嫌疑人犯案时腹部遇刺， 意识不明，送医后确认死亡。
嫌疑人在抢走一块价值6280万日元的高级手表后逃走，警方在约2个半小时后于关西机场发现其踪迹，并以涉嫌“强盗杀人未遂”逮捕。警方认为其当时企图出境。
犯案凶器在珠宝店附近的花草丛中被找到。因受害者已死亡，警方正以涉嫌“强盗杀人”的方向进行调查。
