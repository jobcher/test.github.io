苹果公司表示诉讼威胁到了使苹果公司在激烈竞争市场中脱颖而出的准则。如果诉讼成功，将阻碍苹果公司提供软硬件和服务交叉技术的能力。苹果公司将进行辩护。

广州网红盘大降价引业主不满 项目工作人员称“就如买股票也有涨跌”

据每经，近期，广州黄埔区科学城网红盘翔龙天汇降价吸客刷爆朋友圈。但与此同时，由于项目降价销售，引发部分老业主不满，有人前往售楼部要求退房、退款或其他补偿方案。不过，据克而瑞统计观察，在该网红盘的带动下，当周黄埔区商品住宅网签面积1.1万平方米，环比第九周上涨83.3%。3月21日，记者以购房者身份咨询项目情况，售楼处工作人员回应表示，降价是市场行为，就如买股票也有涨跌。

韩国医科大学教授协会：25日提交辞呈并将工作时间缩短至每周52小时

韩国全国医科大学教授协会非常对策委员会21日举行记者会，公布了协会20日晚与大韩实习住院医师协会、大韩医科大及医学专门研究生院学生协会举行的在线会议的结果。韩国全国医科大学教授协会方面决定，从本月25日起，全国医科大学教授协会的教授们将向所属大学提交辞呈，把工作时间减少到法定的52小时；从4月1日开始，将主要参与急救和重症患者的治疗，尽量减少门诊治疗。

Shein 将向全球品牌开放自己的供应链系统

在中国创立的希音 (Shein) 已经凭借开创先河的小批量生产模式打造出一个低价时装王国。而现在，希音正计划向全球品牌和设计师开放这一模式。希音的执行主席唐伟在一封致投资者的信中宣布了这一计划，并将新举措称为“供应链即服务”。根据这个新计划，希音将向外部品牌和设计师提供希音的供应链基础设施和技术，允许他们利用希音的系统来小批量测试新的时尚产品，并跟踪这些产品在消费者中的受欢迎程度。
