经济学人：为什么大多数乌克兰人开始希望结束战争，甚至一半的人同意放弃一些领土？

根据《经济学人》每日战争追踪器的数据，过去30天，俄罗斯士兵在乌克兰夺取了约645平方公里的领土。
随着战争推进，乌克兰人对战争的疲劳感日益加重。根据盖洛普11月19日和20日发布的调查数据，现在有52%的乌克兰人支持尽快通过谈判结束战争，而一年前这一比例仅为27%。
坚持“战斗到胜利”的比例，从2023年的63%下降到38%。乌克兰总统泽连斯基也改变了说法，最近表达了对明年通过外交途径解决问题的希望。
这种变化，与特朗普在美国重新当选总统有关——他承诺将在24小时内结束战争。

根据路透社报道，俄罗斯战地记者率先披露了这一消息，随后这一消息得到匿名人士证实。乌克兰军方发言人表示尚无相关消息提供。在伦敦，英国首相斯塔默表示不会发表任何相关的评论。
