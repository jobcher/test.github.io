欧洲核子研究中心CERN的LHCb项目3月24日发表成果，首次认定了重子CP破坏的明确迹象。通过分析对撞机过往数据，统计八万多个事件，发现Λb子与反Λb子的衰变产率相差(2.45±0.47)%，以5.2个标准差的置信度认定二者有别。
重子的C（电荷）P（空间反演）对称性破坏是宇宙物质多于反物质的可能成因。本实验结果可为检验标准模型（CKM机制）能否正确刻画CP破坏现象提供依据。
（CERN）
