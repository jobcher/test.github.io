习近平召集民营科技领袖座谈 释放政策转向强信号

2025年2月17日，中国国家主席习近平在北京罕见召开民营企业家座谈会，阿里巴巴创始人马云、比亚迪董事长王传福、宁德时代创始人曾毓群等数十位科技领军人物悉数出席。此次高规格会晤被视为中国政府对民营经济立场的关键转折，明确释放「监管收紧周期结束」与「科技自立需民企支撑」的双重信号。

经历四年多对互联网平台、资本无序扩张的整顿后，习近平在会议上强调民营企业「对经济增长和科技突破不可或缺」，要求政商「坚定信心、防范资本外流」，并点名要求企业加速半导体、人工智能等战略领域攻关。值得注意的是，自2020年因批评金融监管招致打压后鲜少现身的马云位列首排，其旗下企业股价当日显著反弹。分析认为，此举既为对冲新一轮美中贸易战压力（特朗普政府近日宣布加征10%关税），亦旨在缓解因地产危机加剧的经济疲软，但舆论关注政策善意能否转化为制度性松绑，或仅是为特殊时期稳定资本的权宜之计。
